Présentation du Stage

6 mois de stage en tant que développeur front-end.
3 projets principaux, principalement intégration de designs
Découverte de l'agence, du workflow, des équipes...

L'Agence : Wondersauce

  • 45 employés
  • 5 millions $ de chiffre d'affaire
  • 3 ans de business
  • Digital Agency : Web, webapps, design, mobile...
  • Agence jeune, avec les avantages et les inconvénients

Projets

  • John Mayer

    Wondersauce a conçu un site sous forme d'aggrégateur des réseaux sociaux utilisés par JM. Les posts Twitter, Tumblr, Instagram, sont enregistrés dans une BDD, et l'application Angular structure les données JSON récupérées au chargement de la page.

    • Découverte du workflow
    • Node JS, Angular JS
    • Animations CSS
    jm.brewery.ws

  • Carrabba's

    Partie Front-end gérée en interne, back-end par le client via le CMS Sitefinity. Contraintes de structure, nombreux templates de page, modularité du code ont été les caractéristiques de ce projet sur 3 mois.

    • SASS, jQuery
    • Contraintes CMS
    • Handlebars / Partials
    qa.carrabbas.com

  • Bonefish Grill

    Nombreux "goodies" visuels : parallaxe, 'frosted glass', animations, overlays... Challenge en termes de performance. Fonctionnalités avancées : réservation, code front-end découpé en modules.

    • Architecture de projet
    • Javascript, animation (CSS & JS)
    • Communication entre les équipes
    dev.bonefishgrill.com
